Why pair program?

Pair programming is the fine art of programming with another fine human being. It makes programming both more enjoyable as well as more educational. Programming can be lonely if done exclusively by oneself, and having another voice in the void of cyberspace can make time pass more quickly. In addition, pair programming can help a hacker discover unknowns unknowns as well as help her/him understand concepts through explaining it to another person.

What is a navigator?

A navigator is a person in a pair programming pair that dictates the features and structure of a program. In a way, a driver helps direct the macro strategy and direction of the program as it is being built.

What is a driver?

A driver actually does the typing within a program. Going along with our previous analogy, a driver directs the micro nitty gritty of the exact syntax and code format of a program.
